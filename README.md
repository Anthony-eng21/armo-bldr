## Controls

### Placing Bricks

**Mouse**

- Click to add and stack Armo Bricks

**Keyboard**

- Use `w` `a` `s` `d` `q` `e` for ghosting / preview
- `enter / return` key confirms placement
- `R` flips the brick orientation (swaps footprint values)

### Panel

---

**Brick Dimensions**

- Change Armo Brick depth and width in the control panel (top right corner)

**Colors**

- Set an Armo's color with the control panel color picker
